# Show do BITCOIN 

## Apresentação 
- Este projeto se baseia no famoso programa SHOW DO MILHãO
- Consiste em um jogo de perguntas e respostas que contém 10 perguntas com 4 alternativas cada
- O premio total equivale 1 Bitcoin, que é dividido nas 10 perguntas, 0,1 Bitcoin cada pergunta
- O jogador conforme for avançando vai coletando até atingir a pontuação máxima
- Caso o jogador erre a pergunta, receberá 10% do prêmio coletado
- A cada Pergunta o jogador terá a opção de Desistir e receberá 50% do valor coletado
- O jogador terá direito a 3 pulos, que alteram a pergunta mas não o nível 

## Como funciona
1. Instale o projeto na sua máquina
2. Instale as dependencias do projeto através de:
    - pip install -r requirements.txt
3. Execute o arquivo.py no terminal
4. Para iniciar o progama novamente execute o arquivo.py novamente
5. Aproveite
